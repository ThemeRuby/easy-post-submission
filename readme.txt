A plugin that lets users submit posts and manage profiles from the front end. Ideal for news, magazines, and creative platforms.

== Description ==
Enable users to submit posts and manage profiles from the front-end of your site. Ideal for news, magazines, and creative platforms to collect ideas and contributions easily.


== External Services ==

This plugin connects to an API to obtain reCAPTCHA information. It is needed to verify humans when submitting and to avoid spam.
The plugin will connect to reCAPTCHA each time the visitor performs a verification action.
For more details, refer to the reCAPTCHA Privacy Policy: https://policies.google.com/privacy


== Installation ==

To Easy Post Submission, follow these steps:

1. Go to Plugins > Add New.
2. Search for "Easy Post Submission."
3. Install and activate Easy Post Submission from your Plugins page.
4. Navigate to Dashboard > Easy Post Submission.
5. A "Get Started" panel will appear. Choose the default pages and form to import.
6. Click "Complete" to finish the setup.
7. Read the documentation to get started: https://docs.themeruby.com/ruby-submission/
